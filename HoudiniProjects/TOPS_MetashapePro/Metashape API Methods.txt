RunScript.py

1. Install Python 3.11.

2. Download the Metashape Python Module from:
https://www.agisoft.com/downloads/installer/

3. The Python Module is installed in the command prompt/terminal using:
pip install Metashape-*.whl

4. Run the script
"C:\Program Files\Agisoft\Metashape Pro\metashape.exe" -r "RunScript.py"

import Metashape

# List all of the Metashape API methods
print(dir(Metashape))
